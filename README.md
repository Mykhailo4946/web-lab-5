# lab-5

## Виконані вправи
1. **JSON ECHO** (`json_echo.js`) - Парсинг тіла запиту, перевірка на валідність JSON та відправка отриманого об'єкта назад клієнту.
2. **JSON OBJECT** (`json_object.js`) - Отримання об'єкта, валідація типів даних (`string`, `number`), генерація привітання та булевого прапорця віку.
3. **JSON ARRAY** (`json_array.js`) - Робота з масивами у JSON: перевірка елементів масиву, підрахунок суми через `reduce`, визначення середнього значення та обробка порожніх масивів.
4. **JSON CALC** (`json_calc.js`) - Імплементація міні-калькулятора через API. Використання конструкції `switch` для визначення математичної дії, обробка помилок (наприклад, ділення на нуль або невідома операція).
5. **JSON NESTED** (`json_nested.js`) - Робота з глибоко вкладеними об'єктами. Безпечна перевірка вкладених структур перед доступом до них, пошук елементів у масивах (`includes`).
